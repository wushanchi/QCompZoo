# 03-Products / 产业与产品

本目录收录量子计算产业链相关信息，包括主要公司介绍、商用量子计算机型号、核心技术参数等。

## 主要公司

- **IBM Quantum** — 超导量子计算领导者
- **Google Quantum AI** — Sycamore 处理器
- **Rigetti** — 超导量子计算
- **IonQ** — 离子阱量子计算
- **Quantinuum** — 离子阱量子计算综合平台
- **Pasqal** — 中性原子量子计算
- **QuEra** — 中性原子量子计算与传感
- **PsiQuantum** — 光量子计算
- **本源量子** — 中国量子计算公司

## 分类说明

- **公司介绍** — 公司背景、核心业务、技术团队
- **商用产品** — 量子计算机型号、规格参数、量子比特数
- **技术参数** — 产品性能对比表、量子体积(QV)、保真度等
