# 02-Application / 技术应用

本目录收录量子计算的工作原理、系统设计、工程实现方案等技术应用相关内容。

## 分类说明

- **工作原理** — 量子比特的物理机制、量子门操作原理
- **系统设计** — 量子处理器架构、量子算法设计、纠错方案
- **工程实现** — 硬件实现、量子编程、算法优化

## 文件命名规范

`[年份]-[作者/机构]-[标题].pdf`

## 资料索引

- **超导量子计算** - IBM、Google 采用的量子比特技术路线
- **离子阱量子计算** - IonQ、Quantinuum 采用的量子比特技术路线，全连接架构
- **中性原子量子计算** - Infleqtion、Pasqal、QuEra 采用的可扩展技术路线
- **光量子计算** - PsiQuantum 采用的技术路线

### 错误纠正与容错

- **量子化学** - 分子模拟、药物研发
