## 🔢 关于量子计算

量子计算是利用量子力学原理进行信息处理的技术。典型应用包括：
- 量子算法（Shor、Grover、变分量子算法等）
- 量子编程框架（Qiskit、Cirq、PennyLane）
- 量子硬件（超导、离子阱、光量子等）
- 量子机器学习
